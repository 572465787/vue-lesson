##vue项目 
1. vue-cli3.0的安装  
-g global 全局  
````
npm install -g @vue/cli 
或者 
yarn global add @vue/cli
```
2.创建项目
```
  vue ui  可视化界面
  vue create '项目名'  (命名行) 空格是选中 
  baebl router css/less/sass 
```
3.路由懒加载
```
 ()=>import('./views/Home.vue')
```
4.创建目录结构 
   -mock 数据(不是必须有)
   -src
     -components 公共组件 
	 -api   放ajax请求 
	 -views 放页面级组件 
	 -util  工具类
